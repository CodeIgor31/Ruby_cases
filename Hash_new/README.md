# Задание

Напишите функцию get_adjacency_matrix(), которая строит упрощенный вариант матрицы смежности из друзей человека в социальной сети.<br>

Функция принимает массив элементами которого является тоже массивы:<br>

[[Vasy, Pety],<br>
 [Vasy, Oksana],<br>
 [Vasy, Egor],<br>
 [Egor, Pety],<br>
 [Egor, Georgiy],<br>
 [Oksana, Andrey]]<br><br>

Вложженый массив состоит из: имени человека - первый элемент и имени его друга - второй элемент.<br>

Функция должна вернуть хеш, ключем которого будет имя человека, а значением - массив из его друзей.<br>

{<br>
  'Vasy': ['Pety', 'Oksana', 'Egor'],<br>
  'Egor': ['Pety', 'Georgiy'],<br>
  'Oksana': ['Andrey']<br>
}<br>
